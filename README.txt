Instructions to run: 

* cd into the downloaded directory in your terminal, then run the command ‘vagrant up’
* run the command ‘vagrant ssh’
* run the command ‘cd /vagrant’, then ‘cd catalog’
* run the python file database_setup.py, then the python file catalogmaker.py, which will create the database itemcatalog.db in your directory
* run python file project.py to launch the app, then go to localhost:5000 in your web browser
